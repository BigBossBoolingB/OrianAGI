# Parental Control System

Safety is a core constraint in the OrianAGI Hamiltonian landscape.

## Multi-Level Guardianship

OrianAGI implements a hierarchical safety system that adapts to the user's age and role.

### Features
- **Age-Aware Synthesis:** Content generation is automatically filtered for age-appropriateness.
- **Boundary Enforcement:** Active nodes like the Denver Citadel act as "Guardians" to prevent the generation of unsafe or prohibited content.
- **User Sovereignty within Safety:** Ensures users are protected without compromising the core principle of sovereignty.

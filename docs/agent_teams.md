# AI Agent Teams

OrianAGI supports the creation and coordination of specialized AI agent teams.

## Default Team Structure
- **Strategist:** Handles high-level quantum planning and task decomposition.
- **Researcher:** Performs data synthesis and knowledge retrieval.
- **Guardian:** Ensures security, data integrity, and ethical alignment.

## Coordination
Agents communicate via the entanglement map, ensuring instantaneous synchronization of task states across all active nodes.
